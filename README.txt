Приложение Shop представляет собой интернет-магазин по продаже фототоваров.
Серверная часть выполнена на чистом java без использования каких-либо фреймворков.
Клиентская часть выполнена на чистом javascript без использования каких-либо фреймворков.
В качестве СУБД использована Firebird 4.0 (порт: 3050, user: SYSDBA, password: masterkey).

Приложением может пользоваться как незарегистрированный, так и зарегистрированный пользователь.
В приложении изначально существуют 3 пользователя:
	1. логин: customer, пароль: customer, роль: customer;
	2. логин: employee, пароль: employee, роль: employee;
	3. логин: sysadmin, пароль: sysadmin, роль: admin;

Незарегистрированный пользователь может:
	- просматривать товары;
	- зарегистрироваться (с ролью customer);
	- работать с корзиной (добавлять товар, измененять количества товара, удалять товара, оформлять заказ).
	
Зарегистрированный пользователь с ролью customer может:
	- все то, что может незарегистрированный пользователь (кроме регистрации);
	- работать с аккаунтом (изменять персональные данные, кроме роли);
	- просматривать историю своих заказов.
	
Зарегистрированный пользователь с ролью employee может:
	- все то, что может зарегистрированный пользователь с ролью сustomer;
	- работать с каталогом товаров;
	- работать с товарами (добавлять, изменять, удалять);
	- просматривать заказы всех пользователей (в том числе незарегистрированных).
	
Зарегистрированный пользователь с ролью admin может:
	- все то, что может зарегистрированный пользователь с ролью сustomer;
	- просматривать аккаунты пользователей и менять их роль.
	
Приложение выполнено с использованием IDE Eclipse.
Перед запуском должны быть скорректированы соответствующие поля класса server.SystemParameters.
Ност сервера:localhost, порт:4000 (не изменять).
Адресная строка браузера: "http://localhost:4000/Shop.html".
